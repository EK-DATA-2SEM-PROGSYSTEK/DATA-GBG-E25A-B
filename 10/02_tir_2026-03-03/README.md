# Databaser - introduktion til SQL DML & DDL

## Beskrivelse
Vi skal nu til at gemme data i en database fremfor i en ArrayList (eller tekstfil).
Vi skal arbejde med en MySQL database, så det er vigtigt at du har - i det mindste forsøgt - at installere denne inden undervisningen.  Det gælder også MySQL WorkBench, som er det værktøj vi skal bruge til at tilgå databasen. 

I bliver introduceret til sproget SQL som DML(Data Manipulation Language) der bruges til at hente, samt ændre, data i en relationel database.

Desuden skal vi se på hvordan man opretter en database ved hjælp af SQL som DDL (Data Definition Language).
