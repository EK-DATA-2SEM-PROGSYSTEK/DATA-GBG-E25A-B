## Opgave - Collections Framework

### Introduktion
I denne opgave skal der undersøges funktionaliteten i Collections Framework.


### Læringsmål

- At kunne beskrive forskellen mellem Java Collection Framework typer som List, Map og Set
- At kunne udvælge og anvende passende collection typer

### Beskrivelse
Opaven skal laves parvis. En studerende laver følgende undersøgelsen af ```Set``` men den anden undersøger ```Map```.
Efterfølgende skal de forklare deres resultater til hinanden.

Oplysningerne skal findes ved brug af:

[Java API dokumentation Set](https://docs.oracle.com/en/java/javase/21/docs/api/java.base/java/util/Set.html)

[Java API dokumentation Map](https://docs.oracle.com/en/java/javase/21/docs/api/java.base/java/util/Map.html)
