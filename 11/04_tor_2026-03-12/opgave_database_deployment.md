#### Opgave: Database deployment på Azure
1. Følg guiden [MySQL Database Deployment on Azure](azure_mysqlserver_guide.md) til at oprette en MySQL server på Azure.
2. Brug MySQL Workbench til at oprette `world` MySQL database på den oprettede MySQL server.
   Brug [schema](sql/schema.sql) og [data](sql/data.sql) SQL scripts til at oprette tabeller og indsætte data i `world` databasen.
3. Skriv en `select` query i MySQL Workbench for at verificere, at data er korrekt indsat i `countries` tabellen.
